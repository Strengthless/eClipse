Setting up F3 English e-book for RETARDS

This is a guide for retards on how to setup the e-books and get all the answers properly.

However, please note that you shouldn't leak it to anyone else.

Requirements

- Windows OS

- Some braincells

Tutorial

  1. 1. Select the files you want, download them below.
  2. 2. Extract the zip file using WinRAR, or anything that works.
  3. 3. If you're on the TA e-book, run the OUPC_Education.exe file.
    Otherwise, run the e-BOOK.exe file.
  4. 4. Wait for it to load all the dependencies.
